# `cli/mpfr_expr_cli` 设计

## 职责

两种固定 MPFR witness 语法的命令 adapter。

## 数据流

它识别 sqrt 或整数幂输入，把解析与执行委托出去，并渲染单文件汇总。

## 算法与不变量

语法选择属于传输元数据，不是数值启发式；不支持的文件在执行前失败。

## 失败与副作用

文件读取和输出是副作用；MPFR 格式解析与二进制比较留在 `frontend/mpfr_expr`。

## 实现取舍

自动选择语法让命令更短，但有意只支持仓库固定格式。

## 稳定性

本包作为仓库基础设施维护；生成声明可能随 runner 演进，不承诺下游兼容性。
