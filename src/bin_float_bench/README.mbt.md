# `bin_float_bench`

Benchmark-only package for representative public `BinCoeff` operations.

## Contract

It has no application API; use measurements only as dispatch evidence.

## Maintainer Quick Start

```sh
sh tools/run_moon_clean_exec.sh test src/bin_float_bench --target native
```
