# `frontend/gda_expr`

Pure parser and executor for General Decimal Arithmetic `.decTest` documents.

## Contract

It returns explicit diagnostics and case dispositions; filesystem and process concerns remain in CLI/tooling layers.

## Maintainer Quick Start

```sh
sh tools/run_moon_clean_exec.sh test -p Luna-Flow/floating/frontend/gda_expr --target native
```
