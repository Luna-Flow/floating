# `frontend/mpfr_expr`

Oracle adapter for pinned MPFR hexadecimal sqrt, integer-power, and
full-family elementary witnesses.

## Contract

It is not an MPFR binding and accepts only the documented repository grammars.

## Maintainer Quick Start

```sh
sh tools/run_moon_clean_exec.sh test -p Luna-Flow/floating/frontend/mpfr_expr --target native
```
