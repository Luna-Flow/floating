# `frontend/itl_expr`

Parser and executor for the repository-supported ITF1788 line language.

## Contract

Unsupported operations remain explicit and do not imply full IEEE 1788 coverage.

## Maintainer Quick Start

```sh
sh tools/run_moon_clean_exec.sh test -p Luna-Flow/floating/frontend/itl_expr --target native
```
